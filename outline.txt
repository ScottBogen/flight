Start small.

Map-based flight planning software.

Front-end:  React
Back-end:   Express
Database:   MongoDB

Features for version 0.1:
    - Pick a starting or ending airport
    - Pick a distance in miles
    - Generate a result

Features for version 0.2:
    - Implement map, show start and end airport, show straight travel line

Features for version 0.3:
    - Show (limited number of) airports on map, allow user to select node

API Calls:
    - GET airports within a radius
    - GET airports within the 'meat' of the map (0.3)

Front-end display (0.2):
    - Map based
    - User-configurable parameters at top, results pane at bottom
    - Two circles representing min and max range
